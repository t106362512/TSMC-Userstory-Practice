## Acceptance Criteria

### 使用者訂購

實現方式一: 在搜尋車輛時，提供叫車的偏好選項。

- Given: 使用者在使用者模式下使用本應用程式，並閱讀相關條款後授權位置等資訊。
- When: 使用者選擇時段與地點後，送出。
- Then: 使用者等著老司機載他出去放鬆放鬆。
